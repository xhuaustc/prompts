# ✈️ 旅行规划助手

## 完整旅行计划

### 提示词模板
```
我计划去[目的地]旅行，请帮我制定详细的旅行计划。

旅行信息：
- 出发地：[城市]
- 目的地：[城市/国家]
- 出行日期：[开始日期] 至 [结束日期]（共[天数]天）
- 出行人数：[人数和构成，如2大1小]
- 预算范围：[总预算]

旅行偏好：
- 旅行类型：[休闲度假/文化探索/美食之旅/户外探险/购物血拼]
- 住宿偏好：[酒店/民宿/青旅]
- 交通方式：[公共交通/租车/包车]
- 体力要求：[轻松/适中/较强]

请提供：
1. 每日详细行程安排
2. 景点推荐和游玩时长
3. 美食餐厅推荐
4. 交通路线规划
5. 住宿位置建议
6. 预算分配明细
7. 打包清单
8. 旅行注意事项
```

## 周末短途游

```
我想在周末进行一次短途旅行，放松一下。

基本信息：

- 出行方式：[自驾/高铁/其他]
- 预算：[金额]
- 出行人员：[独自/情侣/家庭/朋友]

偏好：
- 想要的氛围：[安静/热闹/小众/网红]
- 活动类型：[自然风光/古镇古城/温泉度假/主题乐园]
- 特殊需求：[带宠物/带老人/带孩子]

请推荐：
1. 3-5个目的地选择
2. 每个地方的亮点
3. 2-3日游简要行程

5. 最佳出行时间
```

## 使用技巧

1. **提前规划**：至少提前1-2个月开始规划，预订更优惠
2. **灵活调整**：保留20%的自由时间，应对意外和即兴发现
3. **本地攻略**：查看当地人推荐，避开游客陷阱
4. **记录分享**：用照片和文字记录，回来后整理成游记
5. **安全第一**：购买旅行保险，保存紧急联系方式
