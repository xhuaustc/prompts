# 🛒 智能购物清单生成器

## 基础购物清单

### 提示词模板
```
我需要采购一周的生活用品，请帮我生成一份购物清单。

家庭成员：[数量和特殊需求]
饮食偏好：[素食/荤食/混合]
特殊需求：[过敏信息/饮食限制]
预算范围：[预算金额]

请按以下分类整理：
1. 新鲜食材（蔬菜、水果、肉类）
2. 主食和调料
3. 日用品
4. 其他必需品

对每项商品标注：
- 建议购买量
- 大致价格
- 保存方法
```

### 使用示例
```
我需要采购一周的生活用品，请帮我生成一份购物清单。

家庭成员：3人（2大1小，孩子5岁）
饮食偏好：荤素搭配，偏清淡
特殊需求：孩子对海鲜过敏
预算范围：500元左右

请按以下分类整理：
1. 新鲜食材（蔬菜、水果、肉类）
2. 主食和调料
3. 日用品
4. 其他必需品

对每项商品标注：
- 建议购买量
- 大致价格
- 保存方法
```


## 使用技巧

1. **提前规划**：每周末规划下周采购清单
2. **检查库存**：采购前先检查家中现有物品
3. **分类整理**：按超市布局分类，提高采购效率
4. **价格对比**：记录常购商品价格，选择最优渠道
5. **灵活调整**：根据实际促销活动调整清单
