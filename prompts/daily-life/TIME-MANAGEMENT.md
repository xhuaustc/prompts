# ⏰ 时间管理与效率提升

## 个性化时间管理方案

### 提示词模板
```
我想提高时间管理能力，请根据我的情况制定方案：

当前状况：
- 身份：[学生/上班族/自由职业/其他]
- 工作时间：[固定作息/弹性工作/轮班]
- 主要困扰：[拖延/时间不够用/效率低/经常加班/无法专注]

- 固定时间段：[上班/上课时间、通勤时间]
- 必做事项：[每天必须完成的任务]
- 个人目标：[短期和长期目标]

生活习惯：
- 精力最佳时段：[早晨/上午/下午/晚上]
- 睡眠习惯：[几点睡几点起]
- 个人兴趣：[需要安排时间的爱好]

请提供：
1. 日常时间分配建议
2. 一周时间规划表
3. 优先级排序方法
4. 提高专注力的技巧
5. 避免拖延的策略
6. 时间追踪和复盘方法
```

## 使用技巧

1. **从小处开始**：不要一次改变所有习惯，选择1-2个优先改善
2. **追踪记录**：使用APP或日记记录时间使用情况
3. **定期复盘**：每周回顾，发现问题并调整
4. **灵活调整**：根据实际情况不断优化方案
5. **奖励自己**：达成小目标时给予自己奖励

## 推荐工具

- 任务管理：Todoist, TickTick, Notion
- 时间追踪：Toggl, RescueTime
- 专注：Forest, Focus@Will
- 日程管理：Google Calendar, Outlook
