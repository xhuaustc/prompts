# 🎨 兴趣爱好发展

## 新兴趣探索

### 提示词模板
```
我想找到新的兴趣爱好：

关于我：
- 年龄/职业：[信息]
- 性格特点：[外向/内向/活跃/安静]
- 当前状态：[工作忙碌/时间充裕/压力大/无聊]

兴趣方向：
- 偏好类型：[艺术/运动/手工/收藏/学习/社交/其他]
- 场地：[室内/户外/都可以]
- 社交性：[独处/社交/都可以]

可投入资源：
- 时间：[每周X小时]
- 预算：[月预算]

期望收获：
- [放松/成就感/社交/技能/其他]

请推荐：
1. 10个适合我的兴趣爱好
2. 每个爱好的：
   - 简介和魅力点
   - 入门门槛（时间/金钱/难度）
   - 预期收获
   - 入门方法
3. 最推荐的3个详细说明
```

## 兴趣入门指南

### 提示词模板
```
我想入门[具体兴趣]：

兴趣：[摄影/绘画/乐器/运动/烹饪/其他]

我的情况：
- 基础：[零基础/有一点了解]
- 目标：[只是玩玩/想达到一定水平]
- 可投入：
  - 时间：[每周X小时]
  - 预算：[金额]

请提供：
1. 入门完整指南
2. 需要的装备/工具清单（含价格建议）
3. 学习路径（从零到入门）
4. 学习资源推荐（课程/书籍/视频/社群）
5. 新手常犯错误和避免方法
6. 3个月入门计划
7. 进阶方向
```
