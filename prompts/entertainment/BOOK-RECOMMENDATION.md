# 📚 图书推荐与阅读

## 个性化书籍推荐

### 提示词模板
```
请根据我的喜好推荐书籍：

阅读偏好：
- 类型：[小说/非虚构/传记/历史/科普/商业/自我提升/其他]
- 风格：[轻松/深刻/专业/通俗/其他]
- 最近喜欢的书：[列出3-5本]

阅读目的：
- [娱乐放松/学习知识/个人成长/工作需要]

阅读能力：
- 阅读速度：[快/中/慢]
- 偏好长度：[短篇/中篇/长篇/都可以]

请推荐：
1. 10本书（含简介和推荐理由）
2. 按优先级排序
3. 标注页数、难度、评分
4. 阅读时长估计
5. 购买/借阅渠道建议
```

## 读书笔记撰写

### 提示词模板
```
请帮我写读书笔记：

书籍信息：
- 书名：[名称]
- 作者：[作者]
- 类型：[类型]

阅读过程：
- 阅读时长：[用了多久]
- 阅读方式：[精读/泛读/跳读]

我的笔记/标注：
[粘贴阅读时的笔记和思考]

请整理成：
1. 书籍核心内容总结
2. 重要观点提炼
3. 金句摘录
4. 个人思考和启发
5. 可实践的要点
6. 推荐指数和理由
```
