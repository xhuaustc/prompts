# 🎬 影视推荐与评论

## 个性化电影推荐

### 提示词模板
```
请根据我的喜好推荐电影：

我的偏好：
- 喜欢的类型：[动作/科幻/爱情/悬疑/喜剧/文艺/恐怖/其他]
- 喜欢的风格：[烧脑/温情/热血/轻松/深刻/其他]
- 最近喜欢的电影：[列出3-5部]

观影场景：
- [自己看/和朋友/和家人/情侣约会]
- 观影时间：[有充足时间/时间有限]

心情/需求：
- [想放松/想思考/想哭/想笑/想刺激]

请推荐：
1. 10部电影（含简介和推荐理由）
2. 按优先级排序
3. 标注年份、时长、豆瓣/IMDB评分
4. 观看渠道提示
5. 如有剧透风险请提醒
```

## 电影观后评论

### 提示词模板
```
请帮我写一篇电影观后感：

电影信息：
- 片名：[名称]
- 导演/主演：[信息]
- 类型：[类型]

我的观影体验：
- 印象深刻的场景：[描述]
- 喜欢的角色：[角色和原因]
- 情感共鸣点：[触动我的地方]
- 个人解读：[我的理解]

评价：
- 整体评价：[评分和简评]
- 优点：[列出]
- 不足：[列出]

写作用途：
- [朋友圈/影评平台/个人日记]

请帮我写出：
1. 引人入胜的开头
2. 剧情简介（无剧透/轻微剧透/全剧透 - 请选择）
3. 深度解析（主题/技法/隐喻）
4. 个人感受
5. 推荐建议（什么人适合看）
```

## 影视主题探索

### 提示词模板
```
我想看[主题]相关的影视作品：

主题：[时间旅行/人工智能/家庭关系/女性成长/其他]

偏好：
- 类型：[电影/剧集/纪录片/都可以]
- 深度：[轻松娱乐/引人思考/深度严肃]
- 地区：[无限制/欧美/亚洲/特定国家]

请推荐：
1. 主题相关的经典必看作品
2. 小众但优质的作品
3. 按主题分类推荐
4. 观看顺序建议（如有关联）
5. 简要剧情和推荐理由
```

## 使用技巧

1. **建立观影清单**：随时记录想看的影片
2. **多元化选择**：不要只看一种类型
3. **观后记录**：写下感受，加深印象
4. **与人讨论**：分享观影体验更有趣
5. **适度观影**：控制时间，避免沉迷
