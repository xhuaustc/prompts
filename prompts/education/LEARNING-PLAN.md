# 📚 学习计划制定

## 新技能学习计划

### 提示词模板
```
我想学习[技能/知识领域]，请帮我制定学习计划：

学习目标：
- 最终目标：[具体想达到什么程度]
- 应用场景：[为什么学/用在哪里]

当前水平：
- [零基础/入门/有一些了解]

可投入资源：
- 时间：[每天/每周可用时间]
- 预算：[课程/书籍预算]
- 学习方式偏好：[看视频/读书/实践/混合]

期望时长：[希望用多久掌握]

请提供：
1. 学习路线图（从入门到目标的阶段划分）
2. 每个阶段的学习内容和资源推荐
3. 具体时间分配（X周计划）
4. 实践项目建议
5. 检验学习成果的方法
6. 常见误区和建议
```

### 使用示例
```
我想学习Python编程，请帮我制定学习计划：

学习目标：
- 最终目标：能独立完成数据分析项目
- 应用场景：工作中的数据分析和自动化

当前水平：
- 完全零基础，没有编程经验

可投入资源：
- 时间：工作日每天晚上1-2小时，周末4-6小时
- 预算：1000元以内
- 学习方式偏好：视频课程+动手实践

期望时长：3-6个月

请提供：
1. 学习路线图
2. 每个阶段的学习内容和资源推荐
3. 具体时间分配（24周计划）
4. 实践项目建议
5. 检验学习成果的方法
6. 常见误区和建议
```

## 使用技巧

1. **SMART原则**：目标要具体、可衡量、可达成、相关、有时限
2. **循序渐进**：由易到难，打好基础
3. **主动学习**：输出倒逼输入，边学边用
4. **定期复习**：遵循遗忘曲线，及时巩固
5. **记录跟踪**：记录学习进度，保持动力

## 学习方法

- **费曼技巧**：教别人是最好的学习
- **番茄工作法**：专注学习，定时休息
- **间隔重复**：spaced repetition提高记忆
- **思维导图**：建立知识结构
- **刻意练习**：走出舒适区，针对性训练
