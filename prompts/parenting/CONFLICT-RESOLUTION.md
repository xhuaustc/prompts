# ⚖️ 冲突处理与问题解决

## 日常冲突快速处理

### 提示词模板
```
两个孩子发生冲突，请提供处理方案：

冲突场景：
- 时间：[刚才/经常发生]
- 地点：[家里/外面]
- 起因：[争抢玩具/打架/争宠/其他]
- 过程：[详细描述发生了什么]

双方表现：
- 姐姐（[X]岁）：[态度/行为/情绪]
- 妹妹（[X]岁）：[态度/行为/情绪]

已采取措施：
- [批评/隔离/讲道理/其他]
- 效果：[有效/无效/反复]

冲突频率：
- [偶尔/经常/每天几次]

请提供：
1. 冲突处理步骤（当下立即处理）
   步骤1：制止冲突（如何做）
   步骤2：情绪安抚（安抚两个孩子）
   步骤3：了解情况（听双方表达）
   步骤4：引导解决（让孩子参与）
   步骤5：达成和解（修复关系）
2. 不同冲突类型的具体话术
3. 父母应避免的错误做法
4. 冲突后的教育时机
5. 预防类似冲突的方法
```

## 使用技巧

1. **保持冷静**：父母情绪稳定是处理冲突的基础
2. **公平公正**：不偏袒，基于事实和规则
3. **听取双方**：让每个孩子都有表达机会
4. **教育为主**：冲突是学习的机会
5. **长远视角**：关注关系而非输赢

## 冲突处理原则

- **安全第一**：防止伤害发生
- **情绪优先**：先处理情绪再讲道理
- **教育意义**：把冲突变成成长机会
- **赋能孩子**：培养自主解决能力
- **关系至上**：维护长期关系比输赢重要
