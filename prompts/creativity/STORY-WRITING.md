# 📖 故事与创意写作

## 短篇故事创作

### 提示词模板
```
请帮我创作一个短篇故事：

故事类型：[科幻/奇幻/悬疑/爱情/现实/恐怖/童话/其他]
故事长度：[字数要求]
目标读者：[儿童/青少年/成人]

核心元素：
- 主题/想表达的：[核心思想]
- 设定：[时代/地点/世界观]
- 角色：[主角特点/配角]
- 冲突：[主要矛盾/挑战]

我的想法：
[如有初步构思]

请提供：
1. 完整的故事大纲
2. 角色设定
3. 故事正文
4. 高潮和转折点设计
5. 意味深长的结尾
```

- 性格特点：[描述]

## 剧情转折设计

- [情节太平淡/缺乏惊喜/读者能猜到]

## 对话写作

### 提示词模板
```
请帮我写一段对话：

场景：[什么情况下]
对话双方：
- 角色A：[性格/目的/情绪状态]
- 角色B：[性格/目的/情绪状态]

对话目的：
- [推动情节/展现性格/制造冲突/揭示信息]

氛围：[紧张/轻松/暧昧/对抗/其他]

请写出：
1. 符合人物性格的对话
2. 潜台词（说的和想的不一样）
3. 对话节奏（快慢/打断/沉默）
4. 动作和神态描写
5. 让对话更生动的技巧
```

## 使用技巧

1. **读者视角**：时刻想着读者想看什么
2. **冲突驱动**：好故事来自冲突和变化
3. **展示不说教**：Show, don't tell
4. **真实细节**：细节让故事可信
5. **反复修改**：好故事是改出来的

## 写作方法

- **雪花法**：从核心扩展到完整故事
- **九宫格法**：规划故事结构
- **SCQA法**：情境-冲突-问题-答案
- **时间轴法**：用时间线梳理情节

## 写作工具

- 灵感记录：印象笔记, Notion
- 大纲规划：Scrivener, workflowy
- 专注写作：小黑屋, OmmWriter
- 素材管理：Pinterest, 鹰眼
- 校对：Grammarly, 火龙果
