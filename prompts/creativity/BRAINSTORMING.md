# 💡 创意头脑风暴

## 产品创意brainstorming

### 提示词模板
```
请帮我头脑风暴新产品创意：

目标领域：[行业/品类]
目标用户：[用户画像]
用户痛点：[现有问题/未被满足的需求]

约束条件：
- 预算：[大致范围]
- 时间：[上市时间要求]
- 技术：[可用技术/团队能力]

创意方向：
- [如有初步想法]

请提供：
1. 15-20个产品创意
2. 每个创意的简要说明
3. 创新点和差异化
4. 可行性初步评估
5. 最有潜力的3个创意详细展开
```

## 使用技巧

1. **量产质变**：先追求数量，再筛选质量
2. **延迟判断**：先发散再收敛，不要过早评判
3. **自由联想**：不设边界，允许疯狂想法
4. **组合创新**：结合两个不相关的概念
5. **反向思考**：如果反着做会怎样？

## 创意方法

- **SCAMPER**：替代/组合/改编/修改/用于其他/消除/重组
- **六顶思考帽**：多角度思考
- **强制联想**：随机词汇联想
- **思维导图**：可视化发散
- **5W1H**：全面提问
