# 📝 会议记录与纪要

## 标准会议记录

### 提示词模板
```
请帮我整理会议记录：

会议基本信息：
- 会议主题：[主题]
- 日期时间：[具体时间]
- 参会人员：[列出姓名和职位]
- 主持人：[姓名]
- 记录人：[姓名]

会议内容：
[粘贴或描述会议讨论的要点，可以是杂乱的笔记]

请整理成：
1. 清晰的会议纪要格式
2. 讨论要点分类梳理
3. 决策事项列表
4. 行动事项（负责人+截止日期）
5. 遗留问题
6. 下次会议安排（如有）
```

## 使用技巧

1. **会前准备**：提前了解议程，准备好记录工具
2. **实时记录**：边听边记关键点，不要等会后凭记忆
3. **分清重点**：重点记录决策、action items、争议点
4. **及时整理**：会后尽快整理，避免遗忘
5. **及时分发**：会后24小时内发送会议纪要

## 会议记录要素

- **决策**：会议做出了什么决定
- **行动**：谁要做什么，什么时候完成
- **讨论**：重要的讨论内容和不同观点
- **问题**：遗留问题和待解决事项

## 记录工具推荐

- 实时笔记：Notion, Evernote, OneNote
- 录音转文字：讯飞听见、Otter.ai
- 思维导图：XMind, MindNode
- 协作文档：Google Docs, 腾讯文档
