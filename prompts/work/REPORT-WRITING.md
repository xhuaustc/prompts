# 📊 报告撰写指南

## 工作周报/月报

### 提示词模板
```
请帮我撰写[周报/月报]：

基本信息：
- 姓名/部门：[信息]
- 汇报周期：[X月X日-X月X日]
- 汇报对象：[直接上级/部门负责人]

本周期工作内容：
1. [项目A]：完成了XXX，进展XX%
2. [项目B]：完成了XXX
3. [日常工作]：XXX
4. [其他]：XXX

数据/成果：
- [具体的数据、指标、交付物]

遇到的问题：
- [如有]

下周期计划：
- [主要工作安排]

请整理成：
1. 清晰的工作汇报格式
2. 重点突出（用数据说话）
3. 问题和计划清晰
4. 语言专业简练
```

## 使用技巧

1. **结构先行**：先列提纲，再填充内容
2. **数据说话**：用具体数字和案例支撑观点
3. **读者导向**：根据读者需求调整详略和专业度
4. **视觉呈现**：用标题、列表、图表提高可读性
5. **三遍打磨**：内容完整→逻辑清晰→语言精练

## 报告写作原则

- **MECE原则**：相互独立、完全穷尽
- **金字塔原理**：结论先行、以上统下
- **5W2H**：What、Why、Who、When、Where、How、How much

## 常用报告结构

1. **执行摘要**：最重要的结论和建议（1页）
2. **背景和目的**：为什么写这个报告
3. **方法论**：如何进行分析/研究
4. **详细内容**：分析过程和发现
5. **结论和建议**：总结和行动建议
6. **附录**：详细数据、参考资料
