# 📧 专业邮件撰写

## 商务邮件模板

### 提示词模板
```
请帮我撰写一封专业的商务邮件：

收件人：[职位/关系]
目的：[请求/通知/跟进/感谢/投诉]
背景情况：[简要说明]

关键信息：
- [要传达的主要内容]
- [需要对方做的事情]
- [时间要求]

语气：[正式/友好/紧急/其他]
邮件语言：[中文/英文]

请包含：
1. 合适的邮件主题
2. 礼貌的开头
3. 清晰的正文（分段，要点明确）
4. 明确的行动呼吁（如需要）
5. 得体的结尾
```

### 使用示例
```
请帮我撰写一封专业的商务邮件：

收件人：潜在客户的采购经理
目的：跟进上周的产品演示会议
背景情况：上周三向对方展示了我们的新产品，对方表示感兴趣，要求发送详细资料

关键信息：
- 感谢对方抽时间参会
- 附上产品详细资料和报价
- 询问是否有其他问题
- 建议安排下一次会议讨论合作细节

语气：专业友好
邮件语言：中文

请包含：
1. 合适的邮件主题
2. 礼貌的开头
3. 清晰的正文（分段，要点明确）
4. 明确的行动呼吁
5. 得体的结尾
```

## 批量邮件/通知

```
需要向团队/部门发送群发通知：

受众：[全员/特定部门/特定群体]
重要程度：[一般/重要/紧急]

通知内容：
- 什么事：[主要信息]
- 何时：[时间]
- 影响：[对收件人的影响]
- 行动：[需要收件人做什么]

请提供：
1. 醒目的主题
2. 简洁的开头（说明通知目的）
3. 要点清晰（用列表、加粗）
4. 明确的行动指示
5. 联系方式（有问题找谁）
```

## 使用技巧

1. **主题明确**：让收件人一眼看出邮件内容和紧急程度
2. **简洁清晰**：避免冗长，用分段和列表提高可读性
3. **校对检查**：发送前检查拼写、语法、附件
4. **及时回复**：尽量在24小时内回复邮件
5. **恰当语气**：根据收件人和情境调整正式程度

## 邮件礼仪

- 使用合适的称呼
- 避免全大写（显得在吼叫）
- 谨慎使用"全部回复"
- 压缩大附件或使用云盘
- 注意时区（国际邮件）
