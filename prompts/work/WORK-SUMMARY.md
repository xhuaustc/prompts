# 📋 工作总结与复盘

## 日常工作总结

### 提示词模板
```
请帮我总结今天的工作：

今日完成：
1. [任务1] - 花费时间、完成情况
2. [任务2] - 花费时间、完成情况
3. [任务3] - 花费时间、完成情况

今日收获：
- [学到了什么/解决了什么难题]

今日问题：
- [遇到的困难/未解决的问题]

明日计划：
- [明天要做的重点事项]

请整理成：
1. 简洁的每日工作总结（100-200字）
2. 关键要点清晰
3. 便于回顾和规划
```

## 项目复盘

### 提示词模板
```
请帮我做项目复盘：

项目基本信息：
- 项目名称：[名称]
- 项目周期：[时间]
- 项目目标：[original goals]
- 团队成员：[核心成员]
- 项目成果：[实际交付]

复盘维度：

1. 目标达成：
   - 原定目标 vs 实际完成
   - 差异原因

2. 过程回顾：
   - 做得好的地方（继续保持）
   - 做得不好的地方（需要改进）
   - 意外和惊喜

3. 数据分析：
   - 时间：计划 vs 实际
   - 成本：预算 vs 实际
   - 质量：目标 vs 实际

4. 团队协作：
   - 协作顺畅的地方
   - 出现问题的地方

5. 经验教训：
   - 可复用的最佳实践
   - 避免重复的错误
   - 对未来项目的建议

请整理成完整的项目复盘报告。
```

## 失败教训总结

### 提示词模板
```
我经历了一次失败/挫折，想从中学习：

事件描述：
- 什么事情：[描述]
- 目标：[原本想达成什么]
- 结果：[实际发生了什么]

失败原因分析：
- 客观原因：[外部因素]
- 主观原因：[自身问题]
- 根本原因：[深层次的原因]

影响：
- 对项目/团队/个人的影响

反思：
- 如果重来会怎么做
- 早期的warning signs
- 应该做but没做的

教训：
- 学到了什么
- 以后如何避免

请整理成：
1. 真诚的反思总结
2. 深入的原因分析
3. 可操作的改进措施
```

## 团队复盘

### 提示词模板
```
请帮我组织团队复盘：

复盘主题：[具体事件/项目/周期]
参与人员：[团队成员]

复盘流程：

1. 回顾目标：
   - 我们想要达成什么？

2. 评估结果：
   - 实际达成了什么？
   - 数据和事实

3. 分析原因：
   - 做得好的地方为什么好？
   - 做得不好的地方为什么不好？

4. 总结规律：
   - 有哪些可复制的经验？
   - 有哪些需要避免的坑？

5. 制定行动：
   - 继续做：[保持的行为]
   - 开始做：[新的行动]
   - 停止做：[要改变的行为]

已收集的信息：
[团队成员的反馈和观察]

请整理成：
1. 完整的团队复盘报告
2. 便于团队共识和行动
3. 建设性的改进建议
```

## 使用技巧

1. **定期复盘**：日/周/月/季/年，形成习惯
2. **及时记录**：趁记忆清晰时总结
3. **客观真实**：不夸大不隐瞒，实事求是
4. **聚焦要点**：抓住20%的关键，避免流水账
5. **行动导向**：总结是为了更好地行动

## 复盘工具

- **PDCA循环**：Plan-Do-Check-Act
- **STAR法则**：Situation-Task-Action-Result
- **4F复盘法**：Fact-Feeling-Finding-Future
- **KPT法**：Keep-Problem-Try
