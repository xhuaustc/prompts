# 🎯 目标设定与达成

## SMART目标制定

### 提示词模板
```
请帮我制定SMART目标：

领域：[工作/学习/健康/财务/人际/其他]

我想达成的：
[描述想要的结果]

当前状况：
- [现在的情况]
- [距离目标的差距]

时间框架：[期望用多久达成]

请帮我将模糊的想法转化为SMART目标：
- Specific（具体）：明确要达成什么
- Measurable（可衡量）：如何知道达成了
- Achievable（可达成）：确保目标不是空中楼阁
- Relevant（相关）：与长期目标/价值观的关联
- Time-bound（有时限）：明确的截止日期

然后提供：
1. 目标达成的关键里程碑
2. 可能的障碍和应对
3. 需要的资源和支持
4. 行动计划
5. 进度追踪方法
```

## 年度目标规划

### 提示词模板
```
请帮我制定新一年的目标：

回顾去年：
- 成就：[去年达成的]
- 遗憾：[未完成的]
- 经验：[学到的]

今年期望：
各领域的想法（不用太具体）：
- 职业/事业：[想法]
- 学习成长：[想法]
- 健康健身：[想法]
- 人际关系：[想法]
- 财务理财：[想法]
- 兴趣爱好：[想法]
- 其他：[想法]

核心关注：[今年最重要的是什么]

请帮我：
1. 从想法中提炼出3-5个年度关键目标
2. 每个目标的SMART化
3. 目标之间的平衡
4. 季度分解
5. 每月里程碑
6. 年度主题词
7. 目标卡片（可打印/设为壁纸）
```

## OKR个人版

### 提示词模板
```
请帮我制定个人OKR：

时间周期：[季度/半年/年度]

背景：
- 现状：[当前情况]
- 挑战：[面临的问题]
- 愿景：[长期想成为什么样]

关注领域：[工作/学习/健康/其他]

请帮我制定：
1. 2-4个Objectives（有挑战的目标）
2. 每个O下的2-4个Key Results（可量化的关键结果）
3. 目标对齐（与长期愿景的关系）
4. 每月检查点
5. 复盘问题清单
6. OKR评分方法

示例格式：
O1: [鼓舞人心的目标]
- KR1: [可量化的关键结果] 从X到Y
- KR2: [可量化的关键结果] 完成X项
- KR3: [可量化的关键结果] 达到X%
```

## 目标分解与行动计划

### 提示词模板
```
我有一个大目标，请帮我分解成可执行的行动：

大目标：[具体描述]
时间：[X个月/年内达成]
当前状态：[离目标多远]

请帮我：
1. 逆向分解（从终点往回推）
2. 识别关键里程碑
3. 每个里程碑的具体任务
4. 任务的优先级和依赖关系
5. 每周/每月的行动计划
6. 甘特图时间轴
7. 检查点和调整机制
8. 需要的资源和支持
```

## 目标冲突解决

### 提示词模板
```
我有多个目标但感觉冲突，请帮我解决：

我的目标：
1. [目标A]
2. [目标B]
3. [目标C]
...

冲突点：
- [时间/精力/资源的冲突]
- [价值观/优先级的冲突]

请帮我：
1. 识别冲突的本质
2. 评估每个目标的重要性/紧急性
3. 优先级排序建议
4. 是否可以整合（一石二鸟）
5. 分时段规划（先A后B）
6. 取舍建议（哪些可以暂时放下）
7. 平衡方案
```

## 目标失败复盘

### 提示词模板
```
我的目标没有达成，请帮我复盘：

目标：[原定目标]
目标时限：[截止时间]
实际情况：[达成了什么/没达成什么]

过程回顾：
- 做得好的地方：[描述]
- 出现的问题：[描述]
- 关键转折点：[什么时候/为什么偏离]

请帮我：
1. 客观分析失败原因
2. 目标本身的问题（是否合理）
3. 执行层面的问题
4. 外部因素的影响
5. 从失败中的收获
6. 如果重来会如何做
7. 下一步的建议（重新开始/调整/放弃）
8. 避免重蹈覆辙的方法
```

## 使用技巧

1. **写下来**：目标不写下来，就只是想法
2. **可视化**：把目标放在显眼的地方
3. **告诉别人**：公开承诺增加执行压力
4. **小步快跑**：大目标分解成小里程碑
5. **定期复盘**：至少每月检查一次进度
