# 💼 职业规划与发展

## 职业生涯规划

### 提示词模板
```
请帮我进行职业生涯规划：

当前状况：
- 年龄/工作年限：[信息]
- 当前职位：[职位]
- 行业/领域：[行业]
- 工作状态：[满意/迷茫/倦怠/其他]

自我认知：
- 优势：[擅长什么]
- 劣势：[不擅长什么]
- 兴趣：[喜欢做什么]
- 价值观：[工作中最看重什么]

职业期望：
- 短期（1-3年）：[期望]
- 中期（3-5年）：[期望]
- 长期（5-10年）：[愿景]

请提供：
1. 职业现状诊断
2. 职业发展路径规划（专业/管理/斜杠）
3. 阶段性目标和里程碑
4. 能力提升计划
5. 行业和公司选择建议
6. 职业转折点应对
7. Plan A和Plan B
8. 行动计划和时间表
```

## 使用技巧

1. **定期复盘**：每半年review一次职业状态
2. **保持学习**：终身学习是竞争力的来源
3. **拓展人脉**：机会往往来自人脉
4. **记录成就**：随时记录工作亮点，备用
5. **适时调整**：根据市场和自身变化灵活调整
