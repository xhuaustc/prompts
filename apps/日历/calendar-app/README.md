# Calendar App (Glassmorphism)

This is a small Next.js + TypeScript + Tailwind CSS calendar UI demo implementing:
- Week view (08:00-16:00) with time-accurate event positioning
- Day/Week/Month view switch
- Event details modal
- Search filtering
- AI assistant popup with typing animation (appears after 3 seconds)
- Hans Zimmer music suggestion with play/pause (simulated)

## Requirements
- Node.js 18+ (recommended)

## Getting Started

Install dependencies:

```bash
npm install
```

Run dev server:

```bash
npm run dev
```

Open `http://localhost:3000`.

## Notes
- Remote background images are loaded from Unsplash via `next/image` remote patterns.
- All code and comments are in English by design.
